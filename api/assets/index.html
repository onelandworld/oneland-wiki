    URL#

    https://api.oneland.world/api/v1/assets?asset_contract_address={address}&offset={offset}&limit={limit}&order_by={orderBy}&order_direction={orderDirection}

    Description#

    Retrieve assets of a contract with specified offset and limit.

    Type#

    GET

    Query Parameters#

    • asset_contract_address: Address of asset contract address.
    • offset: Record offset for the records to return. Default to 0.
    • limit: Limit amount of records to return. Default to 20, and max is 50.
    • order_by: Criteria to sort results. Could only be id or token_id.
    • sort_direction: Sorting direction of the order_by criteria. Could only be asc or desc.

    Sample Curl Command#

    curl -X GET 'https://api.oneland.world/api/v1/assets?asset_contract_address=0x50f5474724e0Ee42D9a4e711ccFB275809Fd6d4a&offset=0&limit=3&order_by=token_id&order_direction=deschttps://api.oneland.world/api/v1/assets?asset_contract_address=0x50f5474724e0Ee42D9a4e711ccFB275809Fd6d4a&offset=0&limit=3&order_by=token_id&order_direction=desc'
