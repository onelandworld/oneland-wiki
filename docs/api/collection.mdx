## URL

```
https://api.oneland.world/api/v1/collection/{slug}
```

## Description
Retrieve more in-depth information about individual collection, includnig contract list.

## Type
GET

## Sample Curl Command
```sh
curl -X GET 'https://api.oneland.world/api/v1/collection/sandbox'
```
