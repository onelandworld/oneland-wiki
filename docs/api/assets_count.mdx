## URL

```
https://api.oneland.world/api/v1/assets/count?asset_contract_address={address}
```

## Description
Retrieve total assets count of a contract.

## Type
GET

## Sample Curl Command
```sh
curl -X GET 'https://api.oneland.world/api/v1/assets/count?asset_contract_address=0x50f5474724e0Ee42D9a4e711ccFB275809Fd6d4a'
```

## Sample Response
```json
{
  "code": 200,
  "message": "success",
  "data": 103243
}
```
